Getting started with LessCss

LessCss is a CSS framework that will help you focus on HTML when creating your website. Build a consistent website and put your atention where it really matters, the content.

Position

There are several ways you can configure the positioning of elements. This could be specially useful fot headers and footers, for example.

  • Sticky
  • Absolute
  • Fixed
  • Relative
    Arrangement
  • Top
  • Bottom
  • Start
  • End

Additionally, you can use arrangement classes along a value to be more precise with positioning. These are some examples:

                                
        <div class="sticky-top"></div>
        <div class="sticky-bottom"></div>
        <div class="sticky-start"></div>
        <div class="sticky-end"></div>
        <div class="sticky top-0"></div>
        <div class="sticky start-0"></div>
        <div class="sticky start-50"></div>
        <div class="sticky top-0 end-0"></div>

Flex

The tradicional flex elements are also available. They work just like in css, but you can organize the content directly from your HTML. You can align text aswell.

  • Justify Content
  • Align-items
justify-content-start
justify-content-center
justify-content-end
                                
        <div class="flex flex-column bg-light-grey w-100 align-items-center p-3">
            <div class="flex flex-row justify-content-start gap-1 w-100 h-10vh">
                <div class="w-25 bg-dark text-light text-start">justify-content-start</div>
            </div>
        
            <div class="flex flex-row justify-content-center my-3 gap-1 w-100 h-10vh">
                <div class="w-25 bg-dark text-light text-center">justify-content-center</div>
            </div>
        
            <div class="flex flex-row justify-content-end gap-1 w-100 h-10vh">
                <div class="w-25 bg-dark text-light text-end">justify-content-end</div>
            </div>
        </div>

Card Title

        </div>

You can use the flex properties to place the image at the top, bottom, end or start of the card. By using the class card-img-position the borders will autocatically be rounded to fit the card shape.
